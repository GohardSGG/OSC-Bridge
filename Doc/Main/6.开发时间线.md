## 6.3 未来规划 (Roadmap)
接手同事可关注以下方向：
1.  **多协议支持**:
    - 考虑支持 MIDI Over Network。
    - 支持 TCP 协议的 OSC (目前主要是 UDP)。
2.  **Mac/Linux 适配**:
    - 目前 `schtasks.rs` 是 Windows 独占，需为 macOS 实现 `LaunchAgent` 逻辑，为 Linux 实现 `systemd` 或 `.desktop` autostart。
3.  **高级过滤规则**:
    - 在转发层增加“地址过滤”或“参数重映射”功能 (Map `/avatar/parameters` to `/param/xyz`)。
4.  **云端控制**:
    - 允许通过公网 HTTP API 控制转发规则 (需注意安全性)。
